📋 方法二：手動從GitHub部署

步驟1: 訪問Railway網站

🌐 打開Railway創建新項目

步驟2: 選擇 "Deploy from GitHub repo"

步驟3: 輸入你的倉庫：Little-pure-light/quantum-wealth-bridge

步驟4: Railway會自動檢測Python應用並開始部署

⚙️ 環境變數設置

部署後，在Railway項目中設置以下環境變數：

DATABASE_URL = (Railway會自動提供PostgreSQL)
SESSION_SECRET = (隨機生成，例如: gigi_quantum_bridge_2024)
SUPABASE_URL = (你的Supabase項目URL)
SUPABASE_ANON_KEY = (你的Supabase匿名密鑰)
SUPABASE_SERVICE_ROLE_KEY = (你的Supabase服務密鑰)
💡 提示： Railway會自動提供PostgreSQL數據庫，你只需要設置其他密鑰即可！

🎯 Supabase密鑰獲取

如果你還沒有Supabase密鑰：

📊 打開Supabase控制台

在Supabase項目中：Settings → API → 複製所需的密鑰
